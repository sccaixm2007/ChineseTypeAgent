# 打字卡牌冒险 🃏✏️

一个专为小学生设计的中文打字练习网页应用。打好字就能抽到宝可梦风格的卡牌！

## 功能特色

### ✏️ 三种学习模式
| 模式 | 说明 |
|------|------|
| 📝 基础模式 | 实时高亮显示正确/错误字符 |
| 拼 拼音模式 | 每个汉字上方显示拼音（约 1500 字覆盖） |
| 🔊 朗读模式 | 打完每句话自动朗读，边打边听 |

### 📊 打字统计
- 实时显示：用时 / 字/分钟速度 / 准确率 / 错误字数
- 完成后展示完整成绩卡
- 易错字统计面板（📊 按钮）

### 🃏 卡牌奖励系统
- 每次完成练习抽一张卡，支持 3D 翻牌动画
- 三种稀有度：普通（70%）· 稀有（25%）· 传说（5%）
- 收集满 3 张升级，满 6 张达到满级
- 支持自定义图片卡池（见下方说明）

### 👤 多用户支持
- 按名字区分存档，数据存于浏览器本地（localStorage）
- 支持积分（⭐）、完成次数、卡牌收藏独立记录

## 快速开始

无需安装任何依赖，直接用浏览器打开即可。

```bash
git clone https://github.com/sccaixm2007/ChineseTypeAgent.git
cd ChineseTypeAgent
open index.html        # macOS
# 或直接双击 index.html
```

## 自定义卡牌

1. 将你的图片放入 `user_define_card/` 文件夹（支持 PNG / JPG / WEBP）
2. 打开应用 → 「🃏 卡牌收藏」→ 底部「🖼️ 配置图片卡池」
3. 每行填写一个路径，例如 `user_define_card/my_card.png`
4. 点击「保存卡池 💾」

建议按稀有度分三类填写，也可以全部填入普通卡池。配置保存在浏览器本地，刷新后依然有效。

系统卡池（`cards/` 目录）同理，可自行添加。

## 文件结构

```
ChineseTypeAgent/
├── index.html            # 主页面
├── style.css             # 样式
├── script.js             # 逻辑（含拼音字典、卡牌数据）
├── user_define_card/     # 放置自定义卡牌图片（图片不含在仓库中）
│   └── README.md         # 使用说明
└── cards/                # 可选：系统卡池图片目录
    ├── common/
    ├── rare/
    └── legendary/
```

## 使用方法

1. 打开页面，输入名字登录
2. 在「打字练习」页粘贴中文课文（需包含句号 。！？）
3. 选择模式，点击「开始练习」
4. 完成后自动抽卡，收集所有卡牌！

## 技术说明

- 纯前端，零依赖，无需网络（朗读功能需浏览器支持 Web Speech API）
- 数据全部存储在浏览器 localStorage，不上传任何信息
- 兼容 Chrome / Safari / Edge 等现代浏览器
